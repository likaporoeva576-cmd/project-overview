### Telegram task bot with voice input

A Telegram bot that helps create and manage tasks from text and voice messages.

What it does:
- creates tasks from simple messages;
- understands dates in Russian;
- works with voice messages;
- sends reminders;
- can integrate with Google Calendar;
- stores tasks in a database;
- has tests and deployment settings.

Tech stack:
- Python;
- Telegram Bot API;
- OpenAI API for voice and command processing;
- PostgreSQL / Supabase;
- Google Calendar API;
- Render for deployment.

My work:

- task parsing;
- reminders;
- calendar integration;
- project structure;
- deployment preparation;
- tests.

Private source code. Public description only.
